Simple event listing plugin

== Description ==

Simple WordPress plugin for events listing. The plugin registers a custom post type `event` with the following meta fields: `Event Date`, `Event Type`, `Event External URL`, `Event Form (Dynamic Fields)`, `Event Banner Image`, `Event Max Attendees`, `Event Current Attendees Count`

== Frequently Asked Questions ==

= Do I need to have a custom post type for the events? =

No. The plugin will create that for you!
